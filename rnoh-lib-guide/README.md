# rnoh-lib-guide

鸿蒙 React Native 第三方库接入指南 - Claude Code Skill

## 功能

查询鸿蒙 RN 第三方库的版本、安装方式、ManualLink 配置、依赖兼容性和注意事项。

## 安装

1. 将此目录复制到 `~/.claude/skills/rnoh-lib-guide/`
2. 克隆 usage-docs 文档库：
   ```bash
   cd ~/.claude/skills/rnoh-lib-guide/
   git clone https://gitee.com/react-native-oh-library/usage-docs.git references/usage-docs
   ```

## 使用

在 Claude Code 中通过 `/rnoh-lib-guide` 调用，或直接询问某个鸿蒙 RN 库的接入方式。
